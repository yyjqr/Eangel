# 热点产品分类调整说明

## 调整概述
将热点产品（机器人、智能硬件、汽车、VR眼镜、军事设备等）的关键词和分类从经济爬虫移至科技爬虫，避免分类混乱。

## 调整内容

### 1. economy_key_config_map.json（经济配置）
**移除的关键词：**
- 纯产品类：机器人、智能硬件、VR/AR/XR、军事设备（导弹、战斗机、武器等）
- 纯技术类：处理器、GPU、智能手机、平板、笔记本、6G等

**保留的关键词：**
- 经济相关：芯片（作为产业）、半导体、电动车市场、汽车销量、新能源产业、电池产业、5G
- 企业名称：比亚迪、茅台（作为企业经济指标）

### 2. tech_key_config_map.json（科技配置）
**新增的产品类关键词：**
- **机器人**：人形机器人、工业机器人、服务机器人（权重1.0-1.3）
- **智能硬件**：智能硬件、物联网、可穿戴设备、智能手表、智能家居（权重0.9-1.1）
- **汽车**：电动汽车、新能源汽车、自动驾驶、无人驾驶（权重0.7-1.2）
- **VR/AR/XR**：虚拟现实、增强现实、混合现实、VR眼镜、Vision Pro、Quest、元宇宙（权重0.9-1.2）
- **军事航天**：武器、无人机、战斗机、雷达、卫星、航天、航空航天（权重0.9-1.1）
- **其他产品**：处理器、GPU、AI芯片、智能手机、平板、笔记本、电池、储能（权重0.7-1.2）

### 3. economy-spider.py（经济爬虫分类）
**移除的分类：**
- 机器人
- 智能硬件
- 汽车与新能源车
- VR/AR/XR
- 军事与国防
- 半导体与芯片
- 新能源

**保留的分类：**
- 宏观经济
- 股市投资
- 企业动态
- 金融科技
- 贸易与供应链
- **产业经济**（新增，包含芯片产业、半导体产业、汽车产业、新能源产业等）
- 经济评论

### 4. main-spider2.py（科技爬虫分类）
**更新的分类：**
- 人工智能（增强关键词：chatgpt、gemini、claude、agi、deepmind）
- **机器人**（新增独立分类）
- **智能硬件**（新增独立分类）
- **汽车与自动驾驶**（新增独立分类）
- **VR/AR/XR**（新增独立分类）
- **军工与航天**（扩展为独立分类）
- **芯片与半导体**（新增独立分类）
- **新能源**（新增独立分类）
- **产品发布**（新增，专门用于产品发布新闻）
- 科学与数学

## 系统架构

### 科技爬虫系统（main-spider2.py + scrapers.py）
- **定位**：科技新闻、产品发布、技术创新
- **数据源**：MIT Tech Review、GitHub Trending、Hacker News、Reddit、Dev.to、Medium、TechCrunch、AI Topics
- **关键词文件**：tech_key_config_map.json
- **重点内容**：AI、机器人、智能硬件、汽车、VR/AR、军工航天、芯片半导体、新能源产品

### 经济爬虫系统（economy-spider.py + scrapers_economy.py）
- **定位**：经济金融新闻、企业财报、市场分析
- **数据源**：Financial Times、Bloomberg、CNBC、The Economist、新浪财经、财新网、东方财富网、凤凰财经、上海证券报
- **关键词文件**：economy_key_config_map.json
- **重点内容**：宏观经济、股市投资、企业动态、金融科技、贸易、产业经济（从经济角度）

## 冲突分析

### 无冲突
- 两个系统使用不同的配置文件
- 两个系统使用不同的爬虫文件
- 两个系统使用不同的数据源
- 两个系统写入同一数据库但分类不同

### 潜在交叉
当某个新闻既涉及产品又涉及经济时（如"特斯拉Q4财报"），会根据来源系统自动归类：
- 如果来自科技爬虫：归为"汽车与自动驾驶"或"企业动态"
- 如果来自经济爬虫：归为"企业动态"或"产业经济"

这种交叉是合理的，因为同一新闻可以从不同角度解读。

## 使用建议

1. **运行科技爬虫**：获取最新的产品、技术、创新新闻
   ```bash
   python3 main-spider2.py
   ```

2. **运行经济爬虫**：获取最新的经济、金融、市场新闻
   ```bash
   python3 economy-spider.py
   ```

3. **数据库查询**：可以通过category字段筛选不同类型的新闻

4. **权重调整**：如需调整某类新闻的优先级，修改对应的json配置文件中的权重值

- ✅ 避免了分类混乱和关键词重复
