## 研究自控机器人的控制和图像传输，识别  
目前基于Arduino，结合超声波传感器，可自主避障。并能够获取摄像头的数据，发送给客户端。   
客户端采用Qt开发，可交互控制自控车。    
后续基于视觉识别，SLAM等进行规划控制。
